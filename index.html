CODI addresses agent-quality imbalance in offline MARL by using LLMs and a compositional diffusion model. It decomposes team-level quality targets into fine-grained agent labels to generate balanced, high-quality trajectory segments for data augmentation. This approach effectively mitigates data imbalance and enables the learning of strong cooperative policies.
